# WikiZone

Projeto criado para estudar e aprimorar minhas habilidades em HTML e CSS.

O site possui uma interface focada em jogos, contendo páginas dedicadas para cada jogo com descrições, imagens e organização em cards.

## Objetivo

Meu objetivo com esse projeto é praticar:
- Estruturação com HTML
- Estilização com CSS
- Organização de páginas
- Responsividade
- Criação de interfaces visuais

## Jogos presentes
- Minecraft
- Stardew Valley
- Left 4 Dead 2

## Tecnologias utilizadas
- HTML5
- CSS
